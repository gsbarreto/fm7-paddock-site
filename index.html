O que esse projeto faz?

O principal objetivo desse projeto é criar um sistema de telemetria móvel para o Forza Motorsport 7, para assim facilitar os treinos e melhorar no jogo. As informações desse projeto são fornecidas pelo seu próprio console (ou PC) e assim convertemos e tratamos os dados para uma HUD. Adicionamos gráficos, e análises telemétricas para uma melhor experiência de jogo.

Então deixa que cuidamos dos dados e você foque na pista!
